# Rust do It - CLI Todo List

Rust do It is a CLI to-do list application that provides a simple and efficient way to manage day-to-day tasks. Inspired by NVIM commands, it keeps the workflow keyboard-first and focused on execution.

## Motivation

When trying to build a productivity system, traditional GUI task tools often introduce too much friction:
"click here to create a task, click there to set a due date, click again to complete it."

For mobile use this is expected, but on desktop these mouse interactions interrupt flow and reduce speed.

## Features

- Keyboard-first task management
- Fast add, edit, and complete commands
- Lightweight local persistence
- Clean terminal interface
