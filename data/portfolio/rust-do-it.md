# Rust do It - CLI Todo List

Rust do It is an CLI to do list application which provides a simple but efficient
way to manage day to day tasks. Being inspired by NVIM commands, the application
seemelessly disapears leaving only what really matters, all the tasks and goals
to be ticked off.

# Motivation

Always when I am trying to make a productivity system, I get frustrated with the 
GUI from traditional todo list Applications: "Click here to enter the name of the
task, click here to seta due date, click here to tick of tasks, ..." 


For Mobile based applications this is rather good and expected, however when
managing tasks on my computer these mouse clicks and movements quickly get in the
way of efficiency, which gets troublesome specially when there is an world of tasks
to be done.

## Features

- Real-time charts and graphs
- User authentication and roles
- Responsive design for all devices
- Export data as CSV or PDF
